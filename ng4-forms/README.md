Simple form with Name and Description fields
- Adds validation to both fields in constructor
- Turn on custom validation for Name field by checking 'Min of 3 chars for Name field' box
- Custom validation is implemented by using Observable on validateCheckbox form field
- Custom validation alert messages for Name field
